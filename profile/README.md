### [`osm-atmdui`](https://github.com/OsmFX-Mods/osm-atmdui) &nbsp;·&nbsp; ATM DUI System

A DUI-based ATM that renders a full banking interface **directly onto ATM screen textures** — not a menu, not an NUI overlay. PIN auth with lockout, withdrawals, deposits, player-to-player transfers, and an interactive physical receipt item.

The UI is built in **React + TypeScript + Tailwind + Framer Motion** and compiles to a standard `html/` folder for FiveM. Framework-agnostic bridge covers QBCore, ESX, and Qbox out of the box.

`HTML` &nbsp; `Lua` &nbsp; `TypeScript` &nbsp; `React`
